"Anaconda3 / Jupyter Notebook 설치 과정 정리"

1. 아나콘다 URL "https://www.anaconda.com/products/individual" 
64bit 다운로드

2.아나콘다 설치 (전 D: 에 했습니다)
D:/tool 폴더에 설치

3. 고급시스템 설정에서 Path 입력 완료
시스템 환경변수 편집 - 환경변수 - 시스템변수 - path -새로만들기
D:\tool
D:\tool\Library\mingw-w64\bin
D:\tool\Library\bin
D:\tool\Scripts

4. 가상환경 설치

Anaconda3 prompt shell (CMD파일)에서
conda create -n cakd3 python=3.8
conda activate cakd3
이후 앞에
(base)에서 (cakd3)으로 변경확인

다음 Jupyter Notebook 설치
conda install jupyter notebook 입력
입력 후 jupyter notebook 실행 확인

jupyter notebook 첫 실행 폴더는 C:\사용자 확인

첫 실행 폴더 변경
jupyter notebook 속성에 들어가서 대상 중 마지막 "~~~" 부분을
"D"\CAKD3\Workspace\CAKD3" 으로 변경
첫 실행 폴더가 Workspace에서 실행되는지 확인

다음 jupyter notebook 패키지 설치

Anaconda3 prompt shell 다시 실행

                    scipy scikit-learn tensorflow keras
